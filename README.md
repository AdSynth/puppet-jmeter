Puppet JMeter
=============

This class installs the latest stable version of JMeter from apache.org.

The init script included in this module is based on the one found in https://github.com/pdrakeweb/puppet-jmeter


Basic usage
-----------

Install JMeter:

    class { 'jmeter': }
